Willkommen bei Adobe Extension Manager 1.8

Branch Test Line 1

Mit dem Extension Manager können Sie schnell und problemlos neue Erweiterungen installieren sowie den Überblick über Erweiterungen behalten, die Sie erwerben, als Teil des Adobe-Registrierungsvorgangs erhalten oder von einer Adobe Exchange-Website heruntergeladen haben. Außerdem unterstützt Sie der Manager beim Bereitstallen von Erweiterungen, indem er die von Ihnen erstellten Erweiterungen automatisch komprimiert und einreicht.

Falls Sie weitere Informationen zu Extension Manager benötigen, Support anfordern möchten oder Ihr Feedback an das Team übermitteln möchten, stehen Ihnen folgende Ressourcen zur Verfügung.


Erste Schritte
Um Extension Manager zu starten, wechseln Sie zu C:\Programme\Adobe\Adobe Extension Manager (WIN) oder Applications\Adobe Extension Manager (Mac), und starten Sie die Anwendung.

Versionshinweise
Die Versionshinweise enthalten aktuelle Informationen sowie Details zu bereits bekannten Problemen im Extension Manager.

Adobe Exchange
Hier können Sie Erweiterungen herunterladen, um Ihre Dreamweaver-, Flash- oder Fireworks-Produkte anzupassen.

Extension Installation File Format In diesem Dokument wird beschrieben, wie Sie ein Erweiterungspaket erstellen, das an andere Benutzer weitergegeben werden kann.

Sie können von Ihnen erstellte Erweiterungen bereitstellen, indem Sie sie auf Adobe Exchange ablegen. Richtlinien zum Testen und Einreichen finden Sie im Hilfedokument Submit to the Exchange.

Dreamweaver Extensibility Newsgroup Diese Newsgroup richtet sich an JavaScript- und C-Entwickler, die Objekte, Verhaltensaktionen, Befehle, Eigenschafteninspectoren, schwebende Paletten sowie Übersetzer für Dreamweaver erstellen, um Erweiterungsmechanismen und Kodiertechniken zu diskutieren.
